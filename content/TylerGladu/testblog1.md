# CS3200 Class Material

So far in CS 3200, I've been building a solid foundation in how databases work, from the theory behind how data is structured to writing real SQL queries. Learning abstract concepts in class and applying them in hands-on work in DataGrip has been the most rewarding part of this course so far and it's cool to see how everything connects in real time. 


# Leuven, Belgium

My favorite part about Belgium so far is the food. I've tried everything from waffles and chocolate to kebab and ramen, and every meal has been incredible. I also enjoyed learning about the European Union and visiting a few of the different institutions. Going to the European Parliament and actually seeing where all 720 members conduct votes was fascinating.
